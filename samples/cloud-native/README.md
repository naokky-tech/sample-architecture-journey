# クラウドネイティブ入門サンプル（Serverless / Kubernetes）

Zenn 連載「フクロウと学ぶアーキテクチャ #5  
クラウドネイティブ（Serverless / Kubernetes）入門」に対応したサンプルコードです。

本ディレクトリには、次の 2 種類の最小構成サンプルが含まれます。

## 1. simple-serverless  
Serverless（AWS Lambda / Azure Functions に相当）の最小構成。

## 2. simple-k8s-api  
Kubernetes 上で動作する最小の FastAPI アプリケーション。
